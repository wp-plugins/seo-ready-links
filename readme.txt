SEO Ready Links is an SEO tool that exports a list of pages and blog post urls in your website for download.

== Description ==

Ready Links is an SEO tool that prints a list of all the pages and blog posts in your website. This list can be used for Linking, BackLinking and cross checking for 404's and redirects. The list can be just a URL list, or a list of html links with a specified anchor text. The uses are limitless. 

1, Enter your keywords. <br />
2, Select Date Range. If empty, it will export all of them.<br />
3, Select Category, Pages etc.<br />
4, Select Export Type. Post by default.<br />
5, Select Export Format. HTML Hyperlink by default.<br />
6, Click Export button.

== Installation ==

1. Upload `ready-links.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
